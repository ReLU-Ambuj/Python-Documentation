# Day 3: Strings and Hashing

## Overview
Day 3 shifts the focus to Strings, string manipulation, character encoding, and introduces Hashing as a means to optimize string queries (like anagram grouping, pattern matching). Since strings in Python are immutable, mastering correct manipulation techniques is important for both performance and correctness.

## Learning Objectives
1. Understand String Immutability in Python and its memory implications.
2. Master ASCII/Unicode relationships via `ord()` and `chr()`.
3. Leverage Hash Maps (dictionaries) and Hash Sets for string processing.
4. Solve 15 core string and hashing drills.

## Structure
- `notes/`: String mutability, character sets, and hashing intros.
- `drills/`: Implementations of core string problems.
- `platform_problems/`: Leetcode style string/hashing problems.
- `speed_coding/`: Drills to write basic string manipulators reflexively.

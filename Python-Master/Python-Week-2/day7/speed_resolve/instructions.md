# Speed Resolve Protocol

## Objective
The purpose of this exercise is to build muscle memory for the specific problems you failed or struggled with during the Day 6 Mock Test.

## Steps
1. Identify the 3 worst performing problems from Day 6.
2. Read the theoretical notes for that pattern.
3. Write the solution on paper or a whiteboard first.
4. Type the optimal solution in under 5 minutes without looking at references.
5. Log your attempts in `tracker.md`.

# Day 7: Mini-Project & Optimization Day

## Overview
Congratulations on reaching the final day of Week 2! Day 7 focuses on solidifying what you've learned. You will refactor previous solutions for optimal Time/Space complexity, resolve problems you struggled with under time pressure, and build a modular Mini-Project.

## Learning Objectives
1. Profile and optimize code for Time (O(1), O(log N), O(N), O(N log N)).
2. Minimize Space Complexity by using in-place operations.
3. Consolidate knowledge by re-solving issues from the Day 6 Mock Test.
4. Architect a simple command-line Mini-Project using proper abstractions.

## Structure
- `optimization_review/`: Tips for cleaning up code and improving efficiency.

# Day 2: Sliding Window Pattern

## Overview
Day 2 focuses entirely on the Sliding Window pattern, which is fundamentally an extension of the Two Pointers approach. This technique is highly effective for reducing $O(N^2)$ (or worse) nested loop solutions to $O(N)$ when dealing with contiguous subsegments (arrays/strings).

## Learning Objectives
1. Understand and implement Fixed-size Sliding Window.
2. Understand and implement Variable-size Sliding Window.
3. Learn window optimization techniques to minimize state tracking overhead.
4. Solve 10 core drills and 15 platform problems.

## Structure
- `notes/`: Concepts and patterns.
- `drills/`: Core repetitive implementations of algorithms.
- `platform_problems/`: Leetcode style problems.
- `repetition/`: Time limits for practicing drills.

# Day 4: Deep Dive into Hash Maps & Sets

## Overview
Day 4 expands on the hashing concepts introduced in Day 3. While Day 3 focused on string hashing, Day 4 covers generalized Hash Maps (dictionaries in Python) and Hash Sets. You will learn about frequency counting, prefix sums with hash maps, and properties of hash sets vs dictionaries.

## Learning Objectives
1. Differentiate between when to use a Hash Map vs a Hash Set.
2. Master frequency counting and grouping patterns.
3. Understand internal mechanics (collision resolution) of Python's hash tables.
4. Solve 15 advanced drills including `O(N)` implementations of Longest Consecutive Sequence.

## Structure
- `notes/`: Theory behind hash map operations and Python specific details.
- `drills/`: Core implementations for hash map patterns.
- `platform_problems/`: Advanced hashing problems on external platforms.
- `review/`: Checklist to review and refactor code.
